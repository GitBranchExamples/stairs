# Пример репозитория GIT состоящего из линии коммитов с одиночными ответвлениями (лестница)


##Правильная визуализация:
   O - - - O - O - O - - - O - - - O - O - - - O

    \_ O       \_ O   \_ O  \_  _ O
    
    
## НЕправильная визуализация:
O - O

    \_ O - O - O
            \_ O - O
                \_ O - O
                    \_ O
